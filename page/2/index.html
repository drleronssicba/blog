Docker常用命令

发表于 2018-09-14

1、更新：sudo apt-get update。
2、安装docker：sudo apt-get install -y docker.io。
3、拉取镜像：docker pull [镜像名称[:版本]] 或 [hub.c.163.com/library/镜像名称[:版本]] 。
4、commit镜像（提交一个新的本地版本镜像）：docker commit [OPTIONS] 【容器id】 [镜像名称[:版本]]。

5、push镜像（将自己本地的镜像上传）：docker push 注册用户名/镜像名 (需要先docker login）。
6、镜像生成容器运行：docker run –name 容器名称 -d -p port1:port2 镜像名称:版本 。（其中，port1是主机端口，port2是容器端口，可映射多个端口）。
7、容器操作：docker start|stop|restart 容器id。
8、主机复制文件到容器：docker cp /主机文件地址 容器id:/容器文件地址。
9、容器复制文件到主机：docker cp 容器id:/容器文件地址 /主机文件地址。
10、进入容器：docker exec -it 容器id bash。
11、删除docker容器和镜像：docker rm 容器id 和 docker rmi 镜像id。
12、查看所有镜像：docker images。
13、查看当前运行的容器：docker ps。
14、查看所有容器：docker ps -a。
15、导出镜像：docker save -o centos7.tar centos。
16、导入本地镜像：docker load –input centos7.tar。

IDEA常用快捷键

发表于 2018-09-14

1.项目跳转：向左跳转ctrl + alt + [，向右跳转ctrl + alt + ]。
2.快捷键查找：ctrl + shift + A。
3.最近文件：ctrl + E。
4.最近浏览：ctrl + alt + 左箭头和ctrl + alt + 右箭头。
5.查找类：ctrl + N。
6.查找文件：ctrl + shift + N。
7.全局查找：ctrl + alt + F。
8.列操作：ctrl + alt + shift + J。
9.live template模板，可自定义。main函数：psvm；打印：sout；public static final String：psfs等。
10.自动校正：alt + enter。
11.重构：shift + f6。
